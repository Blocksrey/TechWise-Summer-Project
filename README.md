# Killem 2D (TechWise Summer Project)
A multiplayer 2D shooty thing written in Python 3

## Dependencies
- Python 3.x (That's it)

## Installation
Example with Pacman:
```sh
pacman -S python38
```

## Execution
Running the game client:
```sh
python killem.py
```

And for the server:
```sh
python killems.py
```
